# invitrek-swt Organization Standards

This repository is the single source of truth for organization-wide coding standards, design guidelines, and best practices for all repositories under the **invitrek-swt** GitHub organization.

## Contents

| Document | Description |

| [Security Policy](SECURITY.md) | How to report security vulnerabilities |

| [Design Guidelines](docs/design-guidelines.md) | Architecture, API, and UI/UX design guidelines |
| [Best Practices](docs/best-practices.md) | Engineering best practices for development, testing, and operations |

## Purpose

Files in this repository are inherited by every repository in the **invitrek-swt** organization. When a repository does not contain its own version of a [community health file](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file), GitHub uses the version from this repository.
