# invitrek-swt

Welcome to **invitrek-swt** — building reliable, scalable, and maintainable software with a focus on engineering excellence.

## Our Principles

- **Quality first** — We write code that is readable, testable, and easy to maintain.
- **Security by design** — Security is considered at every stage of development.
- **Continuous improvement** — We embrace feedback, retrospectives, and iterative refinement.
- **Collaborative culture** — Every voice matters; code reviews are conversations, not judgements.
